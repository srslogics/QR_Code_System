# PayBill Pro

A polished browser-based QR payment and POS billing system for small stores, counters, and service businesses.

## Features

- Dashboard with current bill, paid revenue, pending collection, and invoice series.
- POS billing with searchable catalog, quick add items, custom items, quantity updates, tax/GST, discount, customer details, and cashier tracking.
- Supports UPI payment QR when currency is INR and a UPI ID is entered.
- Standalone QR payment requests for any amount.
- Saved invoice register with paid and pending status.
- Printable receipt with merchant branding and payment QR.
- Copy bill text, download QR image, copy shareable payment details, and print receipt.
- Saves merchant settings, current bill, and invoices locally in the browser.

## Run

Open `index.html` in a browser.

The QR generator is included locally in `vendor/qrcode.min.js`, so it can run without internet after the files are downloaded.
