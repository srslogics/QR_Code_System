# QR POS Billing

A browser-based QR payment and POS billing system.

## Features

- Generate a payment QR for any amount.
- Supports UPI payment QR when currency is INR and a UPI ID is entered.
- Create POS bills with item, quantity, price, tax, and discount.
- Auto-generates printable receipts with a payment QR.
- Download or share payment details.

## Run

Open `index.html` in a browser.

The QR generator uses the QRCodeJS browser library from CDN, so keep the device online when loading the page.
