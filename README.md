# QR POS Billing

A polished browser-based QR payment and POS billing demo for small stores, counters, and service businesses.

## Features

- Generate a payment QR for any amount.
- Supports UPI payment QR when currency is INR and a UPI ID is entered.
- Create POS bills with item, quantity, price, tax/GST, discount, customer details, and payment status.
- Auto-generates printable receipts with merchant branding and a payment QR.
- Quick-add demo products for client presentations.
- Copy bill text, download QR image, share payment details, and print receipt.
- Saves merchant settings and the current bill locally in the browser.

## Run

Open `index.html` in a browser.

The QR generator is included locally in `vendor/qrcode.min.js`, so the demo can run without internet after the files are downloaded.
